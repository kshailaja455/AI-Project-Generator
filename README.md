**AI Project Generator**
An AI-powered web application designed to help students generate academic project ideas based on selected domain and skill level (Beginner, Intermediate, Advanced). The platform simplifies project selection and planning through intelligent recommendations.

**Features**
1.AI-based academic project idea generation
2.Skill-level based recommendations
3.Domain-oriented project suggestions
4.Interactive user interface
5. Real-time AI response generation
6.Full-stack architecture implementation

**Tech Stack:**

**1.Frontend**
React.js
TypeScript
Tailwind CSS
Vite
**2.Backend**
Node.js
Express.js
**3.Database**
PostgreSQL (Neon DB)
**4.AI Integration**
OpenAI API

**Installation & Setup**
1️. Clone Repository
git clone https://github.com/kshailaja455/AI-Project-Generator.git
cd AI-Project-Generator
2️. Install Dependencies
npm install
3. Configure Environment Variables
Create .env file:
DATABASE_URL=your_database_url
OPENAI_API_KEY=your_openai_api_key
PORT=5000
4️. Run Project
npm run dev
Open browser:
http://localhost:5000

**Project Structure**
client/     → Frontend application
server/     → Backend APIs
shared/     → Shared models & schemas

**Purpose of the Project**
This project aims to assist students in selecting suitable academic projects efficiently by leveraging artificial intelligence instead of manual idea exploration.
